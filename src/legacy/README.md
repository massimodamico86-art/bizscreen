# Legacy Code

This folder contains code that has been superseded by newer implementations. Files here are kept for reference only and are **not used at runtime**.

## Contents

- `pages/` - Old page components replaced by new dashboard architecture
- `components/` - Old component implementations
- `hooks/` - Replaced hooks (useDebounce, etc.)
- `data/` - Mock data used during early development
- `utils/` - Old utility functions

## Safe to Delete

All code in this folder can be safely deleted. It is not imported by any active code paths.
