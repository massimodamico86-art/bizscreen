  <action>
    Create `src/components/scene-editor/EmbedWidgetControls.jsx` following the RssWidgetControls pattern exactly.

    **Component signature:** `export function EmbedWidgetControls({ widgetType, props, onPropChange })`

    **Imports:**
    - `{ useState, useEffect }` from 'react'
    - `{ Youtube, Video, Globe, Presentation, WifiOff }` from 'lucide-react' (check icon names match what was used in registry)
    - `{ validateEmbedUrl, extractYouTubeId, extractVimeoId, extractGoogleSlidesId, isPublishedSlidesUrl, getYouTubeThumbnailUrl, getVimeoThumbnailUrl }` from `../../services/embedUtils.js`
    - `{ cacheMedia }` from `../../player/cacheService.js` (for pre-fetching thumbnails into IndexedDB on save)

    **Shared section (all 4 types):**

    State: `urlError` (string|null), `urlWarning` (string|null), `thumbnailPreview` (string|null).

    1. **URL Input** at top of panel:
       - Label with type-specific icon (Youtube/Video/Globe/Presentation) + "Video URL" / "Page URL" / "Slides URL" based on widgetType
       - Text input with placeholder based on type:
         - youtube: "https://youtube.com/watch?v=..."
         - vimeo: "https://vimeo.com/..."
         - webpage: "https://example.com"
         - google-slides: "https://docs.google.com/presentation/d/.../pub"
       - `onChange`: call `onPropChange('url', value)`. Clear errors.
       - `onBlur`: validate using `validateEmbedUrl(url, widgetType)`. Set `urlError` if invalid. Set `urlWarning` if warning (Google Slides non-published URL guidance). Do NOT block saving.
       - Error display: red border on input + red error text below (same pattern as RssWidgetControls)
       - Warning display: yellow/amber text below for Google Slides guidance message

    2. **Inline Thumbnail Preview + Cache Pre-fetch** (below URL input, when URL is valid):

       CRITICAL (Blocker 1 fix): When a thumbnail URL is resolved, the controls MUST:
       (a) Call `onPropChange('thumbnailUrl', resolvedThumbnailUrl)` to store the remote URL in widget props.
       (b) Fetch the thumbnail blob and call `cacheMedia(\`thumbnail:{widgetType}:{resolvedThumbnailUrl}\`, blob)` to pre-cache it in IndexedDB for offline use. Wrap in try/catch — caching failure is non-critical.

       Create a helper function `cacheThumbnailBlob(widgetType, thumbnailUrl)`:
       ```javascript
       async function cacheThumbnailBlob(widgetType, thumbnailUrl) {
         if (!thumbnailUrl) return;
         try {
           const res = await fetch(thumbnailUrl);
           if (!res.ok) return;
           const blob = await res.blob();
           await cacheMedia(`thumbnail:${widgetType}:${thumbnailUrl}`, blob);
         } catch { /* silent — thumbnail caching is non-critical */ }
       }
       ```

       Per-type thumbnail resolution:

       - **YouTube:** Synchronous. Call `getYouTubeThumbnailUrl(videoId)` to get the URL. Show as small preview image (max-height 80px, rounded, object-fit cover). Call `onPropChange('thumbnailUrl', thumbnailUrl)` and `cacheThumbnailBlob('youtube', thumbnailUrl)`.

       - **Vimeo:** Async. In a `useEffect` keyed on `[props.url, widgetType]`, call `getVimeoThumbnailUrl(url)`. On success, set local `thumbnailPreview` state, call `onPropChange('thumbnailUrl', thumbnailUrl)` and `cacheThumbnailBlob('vimeo', thumbnailUrl)`. Show preview image same style as YouTube.

       - **Google Slides (Blocker 2 fix):** When `isPublishedSlidesUrl(url)` is true, extract presentation ID and construct thumbnail URL: `https://docs.google.com/presentation/d/${presentationId}/export/png?pageid=p`. In a `useEffect`, attempt to fetch this URL (HEAD or GET). If successful (2xx), set as thumbnail preview, call `onPropChange('thumbnailUrl', thumbnailUrl)` and `cacheThumbnailBlob('google-slides', thumbnailUrl)`. If fetch fails (403/404 — common for non-published), fall back to a generic Presentation icon placeholder with "Published presentation" text. When `isPublishedSlidesUrl(url)` is false, show only the guidance message — no thumbnail fetch attempted.

       - **Web Page (Blocker 3 fix — demoted to label+icon fallback):** Show a generic Globe icon placeholder with "Web page" text. Do NOT attempt OG image extraction — this would require a server-side proxy/edge function that does not exist yet. If a future phase adds an `/api/og-image` endpoint, thumbnail fetching can be added here. For now, `thumbnailUrl` remains empty string for web pages, and the offline fallback will show the centered "Web Page" label text (which EmbedOfflineFallback already handles when thumbnailUrl is falsy).

    **Type-specific options (below shared section):**

    **youtube / vimeo:**
    - Mute toggle: checkbox, `checked={props.muted !== false}`, `onChange => onPropChange('muted', checked)`
    - Loop toggle: checkbox, `checked={props.loop !== false}`, `onChange => onPropChange('loop', checked)`

    **webpage:**
    - Auto-refresh dropdown: `<select>` with options "None" (0), "Every 5 minutes" (5), "Every 15 minutes" (15), "Every 30 minutes" (30), "Every 60 minutes" (60). Value = `props.refreshIntervalMinutes || 0`. onChange calls `onPropChange('refreshIntervalMinutes', parseInt(value))`.
    - Zoom slider: `<input type="range">`, min 0.25, max 1, step 0.05. Label shows percentage. Value = `props.zoom || 1`. onChange calls `onPropChange('zoom', parseFloat(value))`.

    **google-slides:**
    - Auto-advance interval dropdown: `<select>` with options "5 seconds" (5000), "10 seconds" (10000), "15 seconds" (15000), "30 seconds" (30000), "60 seconds" (60000). Value = `props.delayMs || 5000`. onChange calls `onPropChange('delayMs', parseInt(value))`.
    - Loop toggle: checkbox, `checked={props.loop !== false}`, `onChange => onPropChange('loop', checked)`

    **Styling:** Follow exact same Tailwind classes as RssWidgetControls:
    - Container: `className="space-y-3"`
    - Labels: `className="block text-xs text-gray-500 mb-1"`
    - Inputs: `className="w-full bg-gray-800 text-gray-300 text-xs rounded px-2 py-1.5 border border-gray-700"` (add `border-red-500` when error)
    - Selects: same classes as inputs
    - Checkboxes: `className="rounded border-gray-600 text-blue-500 bg-gray-800"` in a `<label>` with `className="flex items-center gap-2 text-xs text-gray-400 cursor-pointer"`
    - Error text: `className="text-xs text-red-400 mt-0.5"`
    - Warning text: `className="text-xs text-amber-400 mt-0.5"`
