<task type="auto">
  <name>Task 2: VideoWallSync player hook with bezel compensation</name>
  <files>src/player/components/VideoWallSync.jsx</files>
  <action>
Create the player-side video wall synchronization module with two exports:

**1. useVideoWallSync(wallId, screenPosition, isLeader) hook:**

- wallId: UUID of the video wall
- screenPosition: { row, col } of this screen in the grid
- isLeader: boolean

On mount (if wallId truthy):
- Create Supabase Realtime Broadcast channel: supabase.channel(`video-wall:${wallId}`, { config: { broadcast: { self: false } } })
- Subscribe to channel

If leader:
- Export broadcastSync(state) function that sends: { type: 'broadcast', event: 'sync', payload: { currentIndex, timestamp: Date.now(), itemStartTime } }
- Caller (ZonePlayer or content player) calls broadcastSync every 500ms via useInterval

If follower:
- Listen on channel for 'sync' event
- On receiving sync message: compare currentIndex with local state
- Calculate drift = Date.now() - payload.timestamp
- If drift < 200ms and currentIndex differs from local, call onSyncReceived(payload) callback
- Export onSyncReceived as a settable callback ref

On channel subscribe ('SUBSCRIBED' status):
- Send 'ready' broadcast with { screenPosition, isLeader }

Cleanup: supabase.removeChannel(channel) on unmount

Return: { broadcastSync, onSyncReceived, isSubscribed }

**2. VideoWallTransform component:**

Props: { row, col, rows, cols, bezelGapX, bezelGapY, children }

Renders a div wrapper that applies CSS transform to show this screen's portion of the full content:
- scaleX = cols, scaleY = rows (content spans all screens)
- translateX = -(col * 100) - (col * bezelGapX / cols) percent
- translateY = -(row * 100) - (row * bezelGapY / rows) percent
- transform: `scale(${scaleX}, ${scaleY})`
- transformOrigin: '0 0'
- left/top positioning with the calculated offsets
- overflow: hidden on the wrapper
- width: 100%, height: 100% (fills the screen)

The component wraps children (the normal content player) and applies the transform so only this screen's portion is visible, with bezel gap compensation.

**3. getWallTransform({ row, col, rows, cols, bezelGapX, bezelGapY }) utility:**
- Pure function returning the CSS style object
- Exported for testing and direct use

Import supabase from '../../supabase'. Use useRef for channel, useEffect for lifecycle, useCallback for broadcastSync.
