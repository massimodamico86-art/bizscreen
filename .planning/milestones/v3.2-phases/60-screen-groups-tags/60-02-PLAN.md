**1. Extend `src/services/screenGroupService.js` with three new functions:**

a) `pushPlaylistToGroup(groupId, playlistId)`:
- Validate groupId is provided, throw Error if not.
- Update all `tv_devices` where `screen_group_id` equals `groupId`: set `assigned_playlist_id` to `playlistId` (or null if playlistId is falsy, to clear), set `needs_refresh: true`, set `updated_at: new Date().toISOString()`.
- Select `id` from the updated rows to get a count.
- Return `{ devicesUpdated: data?.length || 0 }`.
- Log with the existing `logger`: `logger.info('Pushed playlist to group', { groupId, playlistId, devicesUpdated })`.

b) `bulkDeleteScreenGroups(groupIds)`:
- Validate groupIds is non-empty array, throw Error if not.
- First, unassign all screens from these groups: `supabase.from('tv_devices').update({ screen_group_id: null }).in('screen_group_id', groupIds)`.
- Then delete the groups: `supabase.from('screen_groups').delete().in('id', groupIds)`.
- Return `{ deletedCount: groupIds.length }`.

c) `bulkAddTagsToGroups(groupIds, newTags)`:
- Validate groupIds and newTags are non-empty arrays.
- Normalize newTags: `.map(t => t.trim().toLowerCase()).filter(Boolean)`.
- For each groupId, fetch current tags, compute union with newTags (deduplicate), update. Use a loop since Supabase doesn't support array append in bulk updates:
  ```js
  const { data: groups } = await supabase.from('screen_groups').select('id, tags').in('id', groupIds);
  const updates = groups.map(g => {
    const existing = g.tags || [];
    const merged = [...new Set([...existing, ...normalizedTags])];
    return supabase.from('screen_groups').update({ tags: merged }).eq('id', g.id);
  });
  await Promise.all(updates);
  ```
- Return `{ updatedCount: groupIds.length }`.

Add all three to named exports AND the default export object.

**5. Implement bulk action handlers:**

`handleBulkDelete`:
```js
const handleBulkDelete = async () => {
  const count = selectedIds.size;
  if (!confirm(`Delete ${count} screen group(s)? Screens will be unassigned but not deleted.`)) return;

  setIsDeleting(true);
  try {
    await bulkDeleteScreenGroups([...selectedIds]);
    showToast?.(`${count} group(s) deleted`);
    setSelectedIds(new Set());
    loadData();
  } catch (error) {
    logger.error('Bulk delete failed', { error });
    showToast?.('Error deleting groups: ' + error.message, 'error');
  } finally {
    setIsDeleting(false);
  }
};
```
