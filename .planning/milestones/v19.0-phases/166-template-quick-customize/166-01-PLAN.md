Create `src/services/svgCustomizeService.js` with these exported functions:

1. **normalizeColor(colorValue)** -- Convert any CSS color (hex, rgb(), named) to lowercase 6-digit hex. Implementation: create an offscreen canvas, set fillStyle, read back. For 'none', 'transparent', return as-is. For url() gradient refs, return as-is.

2. **parseSvgForCustomize(svgString)** -- Parse SVG string using `new DOMParser().parseFromString(svgString, 'image/svg+xml')`. Validate that `doc.querySelector('svg')` exists, throw `Error('Invalid SVG content')` if not. Return the Document object.

3. **extractColors(doc)** -- Walk all elements with `[fill]` or `[stroke]` attributes using `doc.querySelectorAll('*')`. For each element, check `el.getAttribute('fill')`, `el.getAttribute('stroke')`, `el.style.fill`, `el.style.stroke`. Collect values, filter out `'none'`, `'transparent'`, values starting with `'url('`. Normalize remaining values with normalizeColor(). Deduplicate using a Set. Return array of unique hex strings.

4. **extractTextNodes(doc)** -- Find all `text, tspan` elements. Skip tspan children of already-processed text parents (same logic as parseSvgTextElements). For each text element: generate an id (`text-{index}`), capture `el.textContent.trim()`, derive a label from element's `id` attribute or first 20 chars of text content. Return array of `{ id, element, text, label }`.

5. **findLogoElement(doc)** -- Query `doc.querySelector('image#logo, image.logo, image#logo-placeholder')`. If null, fall back to `doc.querySelector('image')`. Return the element or null.

6. **swapColor(doc, targetColor, newColor)** -- Normalize both target and new colors. Walk all elements. For each: if `getAttribute('fill')` normalizes to target, `setAttribute('fill', newColor)`. Same for stroke. Also check and update `el.style.fill` and `el.style.stroke` if they normalize to target.

7. **updateText(element, newText)** -- Set `element.textContent = newText`. Use textContent (NOT innerHTML) to prevent XSS per security requirements.

8. **replaceLogo(doc, logoSrc)** -- Call findLogoElement(doc). If found, set both `setAttribute('href', logoSrc)` and `setAttribute('xlink:href', logoSrc)` for SVG 1.1/2 compat (per Pitfall 3).

9. **serializeSvg(doc)** -- Return `new XMLSerializer().serializeToString(doc)`.

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| SVG text content | User-provided text overrides injected into SVG text nodes |
| Logo file | User-uploaded file used as image source in SVG |
| SVG parsing | Template SVG content parsed via DOMParser |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-166-01 | Tampering | updateText() | mitigate | Use el.textContent = value (never innerHTML) to prevent script injection in SVG text nodes |
| T-166-02 | Tampering | replaceLogo() | mitigate | Only accept URLs from URL.createObjectURL(file) for local previews; Supabase storage domain validation happens at upload time |
| T-166-03 | Tampering | parseSvgForCustomize() | accept | DOMParser in 'image/svg+xml' mode is inherently safe -- script elements are not executed in parsed SVG DOMs |
