
2. **Add menu-board conditional render** inside the `WidgetControls` function, after the countdown block and before the `<ColorPicker>` for Text Color:
   ```jsx
   {widgetType === 'menu-board' && (
     <MenuBoardWidgetControls
       props={props}

     />
   )}
   ```

3. **Add size control for clock, date, clock-date, and weather** — these 4 widget types all have a `size` property in their WIDGET_REGISTRY defaultProps ('medium' default) but no control for it. Add a Size selector inside the WidgetControls function, right after the existing `<ColorPicker label="Text Color" ...>` block at the bottom:
   ```jsx
   {(widgetType === 'clock' || widgetType === 'date' || widgetType === 'clock-date' || widgetType === 'weather') && (
     <div>
       <label className="block text-xs text-gray-500 mb-1.5">Size</label>
       <div className="flex gap-1">
         {['small', 'medium', 'large'].map((size) => (
           <Button
             key={size}
             variant="ghost"
             size="sm"
             onClick={() => onPropsUpdate({ size })}
             className={`flex-1 capitalize ${(props.size || 'medium') === size ? 'bg-gray-700 text-white' : 'text-gray-400'}`}
           >
             {size}
           </Button>
         ))}
       </div>
     </div>
   )}
   ```
   Place this INSIDE the `WidgetControls` function's `<div>`, after the Text Color `<ColorPicker>` and before the closing `</div>` of the WidgetControls section.
