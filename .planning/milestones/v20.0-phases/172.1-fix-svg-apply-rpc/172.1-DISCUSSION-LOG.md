## SVG Content Source

### Q2.1 — Priority order for the slide's design_json.svgContent

| Option | Description | Selected |
|--------|-------------|----------|
| Customized → svg_content column → raise (Recommended) | RPC resolves in order; RAISE EXCEPTION if both are null. | ✓ |
| Customized only — client always sends SVG | RPC requires p_customized_svg NOT NULL; client fetches svg_url when needed. | |
| Customized → svg_content → store svg_url as-is | Fallback stores {svgUrl: '...'} in design_json for editor to fetch. | |

**User's choice:** Customized → svg_content column → raise.
**Notes:** Requires svg_content to be populated for every active template (see Q2.2). Deterministic, surfaces missing seed data immediately.

### Q2.2 — How to handle the 12 seeded rows where svg_content is currently NULL?

| Option | Description | Selected |
|--------|-------------|----------|
| Backfill svg_content in a data migration (Recommended) | Read files under public/templates/svg/** into svg_content. One-time. Idempotent. | ✓ |
| Client-side fetch from svg_url when svg_content is null | QuickCustomizePanel / modal fetches svg_url at runtime. Extra round-trip per preview. | |
| Hybrid — backfill now, keep fetch fallback | Both, so future URL-only templates still work. | |

**User's choice:** Backfill svg_content in a data migration.
**Notes:** Fetch fallback deferred — revisit when a URL-only admin path lands.

### Q2.3 — Is there a 24 vs 12 count mismatch?

| Option | Description | Selected |
|--------|-------------|----------|
| Assume 24 means 12 svg + 12 polotno (Recommended) | ROADMAP phrasing was loose; scope stays on the svg leg regardless. | ✓ |
| Verify count against live DB before planning | Run a SELECT to confirm. | |

**User's choice:** Assume 24 means 12 svg + 12 polotno.
**Notes:** Post-answer verification showed the user's assumption was incorrect but the scope anchor still holds. The real picture: migration 097 seeds 12 no-slug rows, migration 167 seeds 12 slug-bearing rows, neither DELETE's the other (167's ON CONFLICT targets slug — 097 rows have slug IS NULL), so svg_templates actually holds 24 rows pointing to 12 unique static files. Backfill still covers 24 rows with 12 file reads. Captured as D-07 in CONTEXT.md; no re-ask was needed because the scope decision was unaffected.
