# Plan 172.1-02 Summary — Migration 170: clone_svg_template_to_scene RPC

## What was built

`supabase/migrations/170_clone_svg_template_to_scene.sql` — a new SECURITY DEFINER PL/pgSQL function that clones an `svg_templates` row into `scenes` + a single `scene_slides` row atomically. Signature matches migration 168's 3-arg shape:

```sql
clone_svg_template_to_scene(
  p_template_id     uuid,
  p_scene_name      text DEFAULT NULL,
  p_customized_svg  text DEFAULT NULL
) RETURNS uuid
```

## How it works

- **Auth preamble** — `auth.uid() IS NOT NULL` check, raises `'Not authenticated'` otherwise.
- **RLS-scoped template fetch** — `WHERE id = p_template_id AND is_active = TRUE AND (tenant_id IS NULL OR created_by = v_user_id)`. This replicates the 167:39–45 SELECT RLS predicate in-function because SECURITY DEFINER bypasses table-level RLS (the predicate is intentionally replicated here).
- **super_admin parity bypass (D-10)** — if the RLS-scoped fetch missed, a follow-up `SELECT role = 'super_admin' FROM profiles` allows super_admins to fetch the row unrestricted. Mirrors 168:84–89.
- **Missing-template raise** — after RLS + super_admin bypass both fail, raises `'Template not found or inactive'`.
- **D-05 three-level SVG resolution** — `p_customized_svg` (if non-null) > `v_template.svg_content` > `RAISE 'Template has no SVG body'`.
- **Atomic scene INSERT** — `tenant_id = v_user_id`, `name = COALESCE(p_scene_name, v_template.name || ' (Copy)')`, `business_type = v_template.category`, `settings = jsonb_build_object('width', 'height', 'orientation')`, `is_active = true`, returns `id` into `v_new_scene_id`.
- **Single scene_slides INSERT** — `position = 0`, `title = v_template.name`, `kind = 'default'`, `design_json = jsonb_build_object('svgContent', v_resolved_svg)`, `duration_seconds = 10`. No slide loop (unlike 168's `FOR v_slide IN SELECT ... LOOP`).
- **GRANT EXECUTE ... TO authenticated** + `COMMENT ON FUNCTION` — standard footer, matching 168.
- **Usage-count increment** intentionally omitted (deferred to Phase 175, mirrors 168:145–147).
