# Requirements: BizScreen

**Defined:** 2026-04-11
**Core Value:** Screens reliably display the right content at the right time, even when offline

## v19.0 Requirements

Requirements for v19.0 Product Gap Fixes. Each maps to roadmap phases.

### Scheduling & Campaigns

- [ ] **SCHED-01**: User can select dayparting presets (Morning, Afternoon, Evening, Late Night) when editing a campaign schedule
- [ ] **SCHED-02**: User can see campaign performance analytics (impressions, play counts, duration) in a dedicated analytics view

### Content Creation

- [ ] **CONT-01**: User can quick-customize a template (brand colors, logo, text overrides) via QuickCustomizePanel without entering the full editor

### Widgets & Data

- [ ] **WDGT-01**: User can view a moderation queue of pending social feed content items
- [ ] **WDGT-02**: User can approve or reject individual social feed content items from the moderation queue

- [ ] **TQAL-04**: CONT-09/CONT-10 "partial" labels resolved in playlists.spec.js
