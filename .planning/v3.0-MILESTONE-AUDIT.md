## External Setup Required

Before Unsplash features work in production:
1. Create Unsplash developer account and application
2. Set API key: `npx supabase secrets set UNSPLASH_ACCESS_KEY=your_key_here`
3. Deploy Edge Function: `npx supabase functions deploy unsplash-proxy`
4. Apply for Unsplash Production access (demo: 50 req/hr, production: 5,000 req/hr)
