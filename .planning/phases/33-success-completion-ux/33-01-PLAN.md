**Imports:**
- useState, useEffect from 'react'

- CheckCircle, ArrowRight, Plus, LayoutGrid from 'lucide-react'
- confetti from 'canvas-confetti'
- Modal, ModalContent, Button from '../../design-system'
- completeUnifiedOnboarding from '../../services/onboardingService'

**Props interface:**
- isOpen: boolean (required) - whether step is visible
- onComplete: function (required) - called after completing onboarding
- onClose: function (required) - called if user closes (same as complete here)
- screenPaired: boolean (optional, default false) - whether screen was paired

**Component structure:**
1. State: completing (boolean) for loading state during API call

2. useEffect on isOpen: When isOpen becomes true, trigger confetti with:
   - particleCount: 100
   - spread: 70
   - origin: { y: 0.6 }
   - disableForReducedMotion: true
   - zIndex: 10001 (above modal overlay)

3. handleGoToDashboard async function:
   - Set completing to true
   - Call await completeUnifiedOnboarding()
   - Call onComplete?.()
   - No need to reset completing since component unmounts

4. Render Modal with:
   - open={isOpen}
   - onClose={onClose}
   - size="md"
   - showCloseButton={false}
   - closeOnOverlay={false}

5. Modal content:
   - Green gradient header accent: "h-2 bg-gradient-to-r from-green-500 to-emerald-600 rounded-t-xl"
   - Centered content with py-8
   - Large green checkmark circle (w-20 h-20 bg-green-100 with CheckCircle icon)
   - Headline: "Your BizScreen is Ready!" (text-2xl font-bold)
   - Conditional subtext based on screenPaired:
     - If true: "Your content is now live on your screen!"
     - If false: "You're all set to create and display amazing content."
   - Primary CTA: Button variant="primary" size="lg" with "Go to Dashboard" text and ArrowRight icon, loading={completing}
   - Secondary actions div with flex justify-center gap-4:
     - "Add More Screens" link with Plus icon (navigate to /devices on click)
     - "Browse Templates" link with LayoutGrid icon (navigate to /templates on click)

**Secondary CTAs behavior:**
- These are text buttons (not full Button components)
- Use same pattern as skip links: "text-gray-500 hover:text-gray-700 flex items-center gap-1"
- On click: call completeUnifiedOnboarding() then navigate (use window.location or import navigate hook)
- Keep it simple: just complete onboarding and navigate
