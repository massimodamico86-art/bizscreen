**Step 2 - Add useDebounce hook inside the file:**

Add a `useDebounce` hook at the top of the file (below imports, above the component). This is a simple, proven pattern already used in TemplatesPage.jsx:
```js
function useDebounce(value, delay) {
  const [debouncedValue, setDebouncedValue] = useState(value);
  useEffect(() => {
    const timer = setTimeout(() => setDebouncedValue(value), delay);
    return () => clearTimeout(timer);
  }, [value, delay]);
  return debouncedValue;
}
```
