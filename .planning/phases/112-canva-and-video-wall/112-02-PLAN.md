  <name>Task 1: Rewire canvaService.js to use Edge Function proxy</name>
  <files>src/services/canvaService.js</files>
  <action>
Rewrite canvaService.js to route all Canva API calls through the canva-proxy Edge Function via supabase.functions.invoke(). Import supabase from '../supabase'.

**Keep unchanged:**
- startCanvaOAuth() - PKCE flow stays client-side (redirect to Canva OAuth page). Update scopes to include 'design:meta:read design:content:read asset:read profile:read' (remove write scopes, add design:meta:read for listing).
- CANVA_TEMPLATE_CATEGORIES array
- openCanvaTemplates(), openCanvaEditor(), openCanvaDesigner(), getCanvaDesignUrl()

**Rewire handleCanvaCallback(code, state):**
- Keep state verification and code_verifier retrieval from sessionStorage
- Instead of direct POST to Canva token URL, call: supabase.functions.invoke('canva-proxy', { body: { action: 'exchange_token', code, codeVerifier, redirectUri: getRedirectUri() } })
- Remove localStorage token storage (tokens now stored server-side by Edge Function)
- Clean up sessionStorage as before

**Rewire isCanvaConnected() to async checkCanvaConnection():**
- Call supabase.functions.invoke('canva-proxy', { body: { action: 'check_connection' } })
- Return boolean from response

**Add listCanvaDesigns({ query, continuation, limit } = {}):**
- Call supabase.functions.invoke('canva-proxy', { body: { action: 'list_designs', query, continuation, limit } })
- Return { items: [...designs], continuation: string|null }

**Add importCanvaDesign(designId, designTitle):**
- Step 1: Call supabase.functions.invoke('canva-proxy', { body: { action: 'export_design', designId, format: 'png' } })
- Step 2: Download exported image from the returned URL via fetch()
- Step 3: Upload to Supabase Storage bucket 'media' with filename `canva-${designId}-${Date.now()}.png`
- Step 4: Create media_asset record in DB with type='image', mime_type='image/png', metadata: { source: 'canva', canva_design_id: designId }
- Return the created media asset

**Add reimportCanvaDesign(designId, existingAssetId):**
- Same as importCanvaDesign but updates existing media_asset record instead of inserting
- Updates url, file_size, updated_at on the existing record

**Rewire disconnectCanva():**
- Delete the canva_oauth_tokens row for current user via supabase.from('canva_oauth_tokens').delete().eq('user_id', userId)
- Remove all localStorage references (TOKEN_KEY, REFRESH_TOKEN_KEY, TOKEN_EXPIRY_KEY)

**Remove:** getCanvaToken(), refreshCanvaToken(), canvaFetch(), createDesign(), getDesign() - all replaced by Edge Function calls.
