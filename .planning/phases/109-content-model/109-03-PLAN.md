2. `validateWorkingHours(workingHours)` -- validates the JSONB shape:
   - Must have keys '0' through '6' (strings, for JSON compatibility)
   - Each entry must have `enabled` (boolean), `start` (string HH:MM), `end` (string HH:MM)
   - If `enabled`, `end` must be after `start`
   - Returns `{ valid: true }` or `{ valid: false, error: 'description' }`

3. `isWithinWorkingHours(workingHours, timezone)` -- checks if the current time is within working hours:
   - If `workingHours` is null/undefined, return true (always on)
   - Get current day-of-week and time in the given timezone using `Intl.DateTimeFormat`
   - Look up today's schedule entry
   - If not enabled, return false
   - Compare current HH:MM against start/end
   - Return boolean

4. `formatWorkingHoursForDisplay(workingHours)` -- returns a human-readable summary like "Mon-Fri 8:00 AM - 6:00 PM" or "Custom schedule"
