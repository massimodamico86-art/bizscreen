   - Label "Daily Target Time"
   - `<input type="time">` with value from `targetTime`
   - `onChange` calls `onPropChange('targetTime', e.target.value)`
   - Same input styling

   - "Full" (Days Hours Min Sec) — active when `unitLabelStyle === 'full'`

   - "Show urgency color" checkbox for `showUrgency`

   - Value from `textColor`
