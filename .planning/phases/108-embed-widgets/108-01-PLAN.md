  <action>
    **Create `src/services/embedUtils.js`:**

    Pure utility module with these functions (no state, no React):

    1. `extractYouTubeId(url)` - Extract YouTube video ID from 5+ URL formats using regex: `youtube.com/watch?v=`, `youtu.be/`, `youtube.com/embed/`, `youtube.com/shorts/`, `youtube.com/v/`. Return 11-char ID or null. Handle query params, hash fragments, trailing slashes.

    2. `buildYouTubeEmbedUrl(videoId, { muted = true, loop = true })` - Construct `https://www.youtube.com/embed/{id}?params`. Always set `autoplay=1`, `controls=0`, `modestbranding=1`, `rel=0`, `playsinline=1`. Set `mute=1/0` and `loop=1/0` from options. CRITICAL: When loop=1, MUST also set `playlist={videoId}` (YouTube requires this for single-video loop).

    3. `extractVimeoId(url)` - Extract Vimeo video ID from `vimeo.com/{ID}` and `player.vimeo.com/video/{ID}`. Return numeric string or null.

    4. `buildVimeoEmbedUrl(videoId, { muted = true, loop = true })` - Construct `https://player.vimeo.com/video/{id}?params`. Set `autoplay=1`, `muted=1/0` (NOT `mute` -- Vimeo uses `muted`), `loop=1/0`, `autopause=0`.

    5. `extractGoogleSlidesId(url)` - Extract presentation ID from `docs.google.com/presentation/d/{ID}/...`. Return ID string or null.

    6. `isPublishedSlidesUrl(url)` - Return true if URL contains `/pub` or `/embed`, false otherwise.

    7. `buildGoogleSlidesEmbedUrl(presentationId, { delayMs = 5000, loop = true })` - Construct `https://docs.google.com/presentation/d/{id}/embed?start=true&loop=true/false&delayms={ms}`.

    8. `getYouTubeThumbnailUrl(videoId)` - Return `https://img.youtube.com/vi/{id}/hqdefault.jpg` (always available, no API key needed). Use `hqdefault` not `maxresdefault` (hqdefault always exists).

    9. `getVimeoThumbnailUrl(videoUrl)` - Async. Fetch `https://vimeo.com/api/oembed.json?url={encoded}&width=640`. Return `thumbnail_url` from response or null on error. No auth needed.

    10. `validateEmbedUrl(url, type)` - Return `{ valid: boolean, error?: string }`. Check: non-empty, valid URL format, type-specific checks (YouTube: extractable ID, Vimeo: extractable ID, google-slides: extractable presentation ID, webpage: valid URL). For google-slides with non-published URLs, return `{ valid: true, warning: "This looks like a regular Slides link. To display it on screen, publish it first: File > Share > Publish to web." }`.

    - If no thumbnailUrl: centered label text in gray
    - Bottom-right badge with `WifiOff` icon (12x12) + "Requires Internet" text, semi-transparent dark background, small text (0.625rem)
