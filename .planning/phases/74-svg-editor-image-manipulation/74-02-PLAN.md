<task type="auto">
  <name>Task 1: Implement crop mode with visual overlay and apply/cancel in FabricSvgEditor</name>
  <files>src/components/svg-editor/FabricSvgEditor.jsx</files>
  <action>
Add crop mode state and handlers to FabricSvgEditor:

1. **State:** Add `const [isCropMode, setIsCropMode] = useState(false);` and `const cropDataRef = useRef(null);` to store the original image reference during crop.

2. **handleStartCrop:** When the user clicks Crop on a selected image:
   - Store reference to the target image object in `cropDataRef.current = { image: obj, originalClipPath: obj.clipPath || null }`.
   - Create a semi-transparent dark overlay rect covering the canvas (for visual dimming).
   - Create an interactive crop region `fabric.Rect` initialized to the image's current visible bounds. Set it with:
     - `fill: 'transparent'`
     - `stroke: '#00ff00'` (green dashed border)
     - `strokeDashArray: [5, 5]`
     - `strokeWidth: 2`
     - `cornerColor: '#00ff00'`, `cornerSize: 10`, `transparentCorners: false`
     - `hasRotatingPoint: false` (no rotation on crop rect)
     - `id: '__crop_rect__'`
   - Position the crop rect to match the image's bounding box (left, top, width * scaleX, height * scaleY).
   - Add the overlay and crop rect to canvas, select the crop rect.
   - Deselect the image (but keep it visible).
   - Set `isCropMode` to true.
   - Disable selection of all other objects while in crop mode.

3. **handleApplyCrop:** When the user confirms the crop:
   - Get the crop rect's final position and size on canvas.
   - Calculate the crop region relative to the image's coordinate space:
     ```javascript
     const img = cropDataRef.current.image;
     const cropRect = canvas.getObjects().find(o => o.id === '__crop_rect__');
     // Convert crop rect bounds to image-local coordinates
     const cropLeft = (cropRect.left - img.left) / (img.scaleX || 1);
     const cropTop = (cropRect.top - img.top) / (img.scaleY || 1);
     const cropWidth = (cropRect.width * (cropRect.scaleX || 1)) / (img.scaleX || 1);
     const cropHeight = (cropRect.height * (cropRect.scaleY || 1)) / (img.scaleY || 1);
     ```
   - Apply a `fabric.Rect` as `clipPath` on the image:
     ```javascript
     img.set({
       clipPath: new fabric.Rect({
         left: cropLeft,
         top: cropTop,
         width: cropWidth,
         height: cropHeight,
         absolutePositioned: false,
       }),
     });
     ```
   - Remove the crop rect and overlay from canvas.
   - Re-enable selection on all objects.
   - Select the image.
   - Set `isCropMode` to false, `cropDataRef.current = null`.
   - Mark `hasUnsavedChanges`.
   - Call `canvas.renderAll()`.

4. **handleCancelCrop:** Discard the crop:
   - Remove crop rect and overlay from canvas.
   - Restore original clipPath if any: `cropDataRef.current.image.set('clipPath', cropDataRef.current.originalClipPath)`.
   - Re-enable selection on all objects.
   - Select the original image.
   - Set `isCropMode` to false, `cropDataRef.current = null`.
   - Call `canvas.renderAll()`.

5. **Guard other interactions:** While `isCropMode` is true, prevent object deletion, deselection of crop rect, and other toolbar actions. Add a check at the top of `handleDeleteObject` and keyboard shortcut handlers: `if (isCropMode) return;`.

Pass `isCropMode`, `onStartCrop={handleStartCrop}`, `onApplyCrop={handleApplyCrop}`, `onCancelCrop={handleCancelCrop}` as props to TopToolbar.
