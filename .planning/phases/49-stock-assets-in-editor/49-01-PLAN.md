**3. Add onDrop handler to canvas container div:**
On the same container div, add an `onDrop` handler:
```jsx
onDrop={(e) => {
  e.preventDefault();
  const jsonData = e.dataTransfer.getData('application/json');
  if (!jsonData) return;

  let asset;
  try {
    asset = JSON.parse(jsonData);
  } catch {
    return;
  }
  if (!asset?.url) return;

  const canvas = fabricCanvasRef.current;
  if (!canvas) return;

  // Convert screen coordinates to canvas coordinates accounting for zoom
  const canvasEl = canvas.lowerCanvasEl || canvas.getElement();
  const rect = canvasEl.getBoundingClientRect();
  const x = (e.clientX - rect.left) / zoom;
  const y = (e.clientY - rect.top) / zoom;

  // Fire download tracking for Unsplash photos
  if (asset.type === 'unsplash' && asset.downloadTrackingUrl) {
    trackDownload(asset.id, asset.downloadTrackingUrl);
  }

  // Load and place image at drop position
  fabric.FabricImage.fromURL(asset.url, { crossOrigin: 'anonymous' })
    .then((img) => {
      const maxWidth = canvasWidth * 0.4;
      const maxHeight = canvasHeight * 0.4;
      const scale = Math.min(maxWidth / img.width, maxHeight / img.height, 1);
      img.set({
        left: x - (img.width * scale) / 2,
        top: y - (img.height * scale) / 2,
        scaleX: scale,
        scaleY: scale,
        id: `${asset.type || 'dropped'}-${Date.now()}`,
        name: asset.name || 'Dropped Asset',
      });
      canvas.add(img);
      canvas.setActiveObject(img);
      canvas.renderAll();
      setHasUnsavedChanges(true);
      syncCanvasObjects();
    })
    .catch((err) => {
      logger.error('Failed to load dropped image', { error: err });
      showToast?.('Failed to load image', 'error');
    });
}}
```

**Key implementation notes:**
- Use `canvas.lowerCanvasEl || canvas.getElement()` to get the actual canvas element for `getBoundingClientRect()` -- Fabric.js 6 uses `lowerCanvasEl` for the lower canvas.
- Divide screen coordinates by `zoom` (the existing zoom state variable) to convert to canvas coordinates. The canvas container uses `transform: scale(${zoom})` CSS.
- Max width/height is 0.4 (40%) of canvas dimensions for dropped images -- slightly smaller than the centered insertion (which uses 0.5) to leave room for positioning.
- `syncCanvasObjects()` and `setHasUnsavedChanges(true)` mirror the existing `handleAddImageFromUrl` pattern.
- The `type` field from drag data is used for the `id` prefix and for Unsplash download tracking detection.
