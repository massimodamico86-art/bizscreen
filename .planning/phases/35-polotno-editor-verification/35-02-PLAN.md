  <name>Task 1: Add change detection in iframe editor (main.jsx)</name>
  <files>scripts/polotno-build/src/main.jsx</files>
  <action>
Add store change listener in the Polotno iframe editor (main.jsx):

1. After store is created and configured, add a change listener:
```javascript
// Track if design has been modified
let hasChanges = false;
let changeTimeout = null;

// Debounced change notification to parent
const notifyChange = () => {
  if (changeTimeout) clearTimeout(changeTimeout);
  changeTimeout = setTimeout(() => {
    if (!hasChanges) {
      hasChanges = true;
      sendToParent('designChanged', { dirty: true });
    }
  }, 500); // Debounce to avoid excessive messages
};

// Listen for any store changes
newStore.on('change', notifyChange);
```

2. Reset hasChanges when design is saved:
In the handleSave function (TopBar), after successful save, add:
```javascript
hasChanges = false;
```

3. Also reset on initial design load:
In the message handler for 'loadDesign' and 'loadTemplate', set:
```javascript
hasChanges = false;
```

This ensures the parent knows when the user has made changes that haven't been saved.
