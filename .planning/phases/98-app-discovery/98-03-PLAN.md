# Plan 98-03: Navigation Map & Route Documentation

## Goal

Compile all discoveries from plans 98-01 and 98-02 into a complete route list document (`screenshots/ROUTE_MAP.md`) that maps every URL path / page ID to its page name, navigation section, authentication requirement, and screenshot reference.

## Context

- Plans 98-01 and 98-02 have already visited every route and captured screenshots in `./screenshots/98-*.png`
- The app has two navigation models:
  - **URL-based routes** (public pages): Defined in `src/router/AppRouter.jsx` -- each page has its own URL path
  - **State-based pages** (authenticated app): Defined in `src/App.jsx` -- pages are rendered by the `pages` object keyed by page ID, switched via `setCurrentPage()`
- The route map document should be exhaustive and structured for use by subsequent audit phases
