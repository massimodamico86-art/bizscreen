# Plan 100-03: Media, Layouts & Templates CRUD Walkthrough

## Goal

Walk through the complete CRUD lifecycle for Media Library, Layouts/Templates, and SVG Templates -- upload/create, browse/filter, edit, and delete -- capturing screenshots at every step.

## Context

- The dev server runs on `http://localhost:5173` with `VITE_DEV_BYPASS_AUTH=true`
- All screenshots go to `./screenshots/` with prefix `100-`
- Use MCP Playwright tools: browser_navigate, browser_take_screenshot, browser_snapshot, browser_click, browser_type, browser_fill_form
- Navigation uses `window.__setCurrentPage('pageId')` for authenticated pages
- **Media Library** (`media-all`, `media-images`, `media-videos`, etc.): Grid/list view, upload via modal/drag-drop, filter by type tabs, search, delete with usage check, bulk selection, folder navigation, pagination
- **Layouts/Templates page** (`layouts`): OptiSigns-style gallery with left sidebar categories (Featured, Popular, Your Templates, by industry), search, template cards that open in Polotno editor modal, "Your Templates" section shows user-created designs
- **SVG Template Gallery** (`templates`): Template cards with skeleton loading, debounced search, collapsible filter sidebar (categories, industries, styles), Featured/Popular/Recent sections, edit/delete user designs, opens SVG editor (`svg-editor-{id}`)
